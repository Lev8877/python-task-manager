Мне нужно помнить что в insert() в начале передается индекс первым аргументом, а вторым уже значение. pop() просто удаляет из массива жлемент по индексу переданный в аргумент. remove() тоже самое только в аргумент передается уже не индекс а элемент массива. index() вернет индекс элемента массива, в эту комманду передается элемент самого массива. Количество определенного значение это значит нужно определить сколько раз определенный элемент входит в определенную коллекцию. Кортеж можно распаковать таким удобным способом: x,y = one где one = (1,2) => x = 1, y = 2. Задание множества через фигурные скобки: sett = {1,2,3,4}. | логический оператор для объединения двух ммножеств в один. & логический оператор для нахождения пересечений элементов двух множеств. - логический оператор для отрицания одного множества из другого. ^ - логический оператор симметрической разницы - надет значения множеств в первом и втором но не их объединения. remove(a) удалит элемент a из множества но вызовет ошибку если такого элемента нет. discard(a) удалит элемент a из множества и не вызовет ошибку даже если такого элемента в коллекции нет. sort() - отсортировывает старый массив и ничего не возвращает, работает только с массивами. sorted() возвращает новый массив из отсортированной последовательности любого типа данных. user.get(a,n) - вернет значение словаря user по ключу a, если такого ключа нет ничего не вернет или вернет n если был передан второй аргемент что безопасней чем просто использовать квадратные скобики для получения значений из словаря. user.pop(a) - удалит ключ a из словаря user. user.keys(), user.values() - вернет массив из ключей/значений словаря user. user.items() - вернет словарь из пар кортежей (ключ,значение). user.copy - создаст копию словаря User, лучше передовать это в переменную. user.clear() полностью отчистит словарь и сделает его пустым. extend() в отличии от append() добавляет в массив не целый объект (например [2,3]) а каждый элемент отдельно из любой последовательности любого типа данных (то есть в начале 2, потом 3 из массива [2,3]).
